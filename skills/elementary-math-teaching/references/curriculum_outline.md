# 小学1-6年级数学课程大纲

> 基于《义务教育数学课程标准（2022年版）》，按人教版主线整理。三个学段：第一学段（1-2年级）、第二学段（3-4年级）、第三学段（5-6年级）。

---

## 一年级

### 数与代数
| 知识点 | 核心内容 | 课标要求 |
|--------|----------|----------|
| 1~20各数的认识 | 数数、读数、写数、数序、大小比较 | 能正确数出20以内物体个数 |
| 20以内加减法（不进位不退位） | 十加几、十几减几 | 正确口算 |
| 20以内进位加法 | 凑十法（9+几、8+几、7+几、6+几） | 熟练口算 |
| 20以内退位减法 | 破十法、平十法 | 熟练口算 |
| 100以内数的认识 | 数位（个位、十位）、读写法、大小比较 | 理解数位概念 |
| 整十数加減整十数 | 以十为单位的加减 | 正确计算 |
| 认识人民币 | 元、角、分，简单换算 | 能进行简单购物计算 |

### 图形与几何
| 知识点 | 核心内容 | 课标要求 |
|--------|----------|----------|
| 认识图形（一） | 长方体、正方体、圆柱、球 | 能辨认和分类 |
| 位置 | 上下、前后、左右 | 能用方位词描述相对位置 |
| 认识图形（二） | 长方形、正方形、三角形、圆 | 能辨认和描画 |

### 统计与概率
| 知识点 | 核心内容 |
|--------|----------|
| 分类与整理 | 按给定标准或自选标准分类，用简单统计表呈现 |

### 综合实践
- 认识钟表（整时、半时）
- 用加减法解决简单实际问题

---

## 二年级

### 数与代数
| 知识点 | 核心内容 | 课标要求 |
|--------|----------|----------|
| 100以内加减法笔算 | 进位加、退位减的竖式计算 | 正确笔算，理解算理 |
| 表内乘法（一）（二） | 1~9乘法口诀，乘法的意义 | 熟练背诵口诀，正确口算 |
| 表内除法（一）（二） | 除法的意义，用口诀求商 | 理解平均分，正确口算 |
| 有余数的除法 | 余数概念、余数<除数 | 理解余数含义 |
| 万以内数的认识 | 千、万，数位顺序表，读写法 | 理解十进制位值 |
| 混合运算 | 两步加减混合，含小括号 | 掌握运算顺序 |
| 克与千克 | 质量单位，1千克=1000克 | 建立质量观念 |

### 图形与几何
| 知识点 | 核心内容 | 课标要求 |
|--------|----------|----------|
| 长度单位 | 厘米、米，1米=100厘米，用尺测量 | 建立长度观念 |
| 角的初步认识 | 角的各部分名称，直角、锐角、钝角 | 用三角尺判断直角 |
| 观察物体（一） | 从不同角度观察简单物体 | 辨认不同方向看到的形状 |

### 统计与概率
| 知识点 | 核心内容 |
|--------|----------|
| 数据收集整理 | 用"正"字法记录，简单统计表，提出问题并回答 |

### 综合实践
- 认识时间（时、分，1时=60分）
- 数学广角——搭配（一）：简单排列组合

---

## 三年级

### 数与代数
| 知识点 | 核心内容 | 课标要求 |
|--------|----------|----------|
| 万以内的加减法（二） | 三位数加减三位数笔算、估算 | 正确笔算，合理估算 |
| 多位数乘一位数 | 口算、笔算（含进位），0的乘法 | 掌握笔算方法 |
| 除数是一位数的除法 | 口算、笔算（含商中间/末尾有0） | 掌握笔算方法 |
| 倍的认识 | 一个数是另一个数的几倍 | 理解倍数关系 |
| 分数的初步认识 | 几分之一、几分之几，同分母加减 | 初步理解分数含义 |
| 小数的初步认识 | 一位小数的读写、大小比较、加减 | 初步理解小数含义 |
| 吨的认识 | 1吨=1000千克 | 建立吨的观念 |

### 图形与几何
| 知识点 | 核心内容 | 课标要求 |
|--------|----------|----------|
| 测量 | 毫米、分米、千米，1千米=1000米 | 建立长度观念 |
| 周长 | 周长的含义，长方形和正方形周长计算 | 掌握公式并应用 |
| 面积 | 面积的含义，面积单位（cm², dm², m²） | 理解面积概念 |
| 长方形和正方形面积 | 面积公式推导与计算 | 掌握公式 S=ab, S=a² |

### 统计与概率
| 知识点 | 核心内容 |
|--------|----------|
| 复式统计表 | 合并单式统计表，读取信息 |

### 综合实践
- 年、月、日（大月、小月、平年、闰年、24时计时法）
- 数学广角——搭配（二）：有序排列、组合
- 位置与方向（一）：东、南、西、北等八个方向

## 四年级

### 数与代数
| 知识点 | 核心内容 | 课标要求 |
|--------|----------|----------|
| 大数的认识 | 亿以内/以上数的读写、改写、求近似数 | 正确读写，理解数级 |
| 三位数乘两位数 | 笔算，积的变化规律 | 掌握笔算方法 |
| 除数是两位数的除法 | 试商、调商 | 正确笔算 |
| 四则运算 | 加减乘除各部分关系，含括号的运算顺序 | 掌握运算顺序 |
| 运算定律 | 交换律、结合律、分配律 | 理解并用字母表示 |
| 小数的意义和性质 | 小数意义、读写法、性质、大小比较、小数点移动 | 理解小数本质 |
| 小数的加减法 | 笔算，小数加减混合运算 | 正确计算 |
| 分数的意义和性质 | 分数意义、分数单位、真/假/带分数 | 深入理解分数 |

### 图形与几何
| 知识点 | 核心内容 | 课标要求 |
|--------|----------|----------|
| 角的度量 | 量角器使用，角的分类（锐/直/钝/平/周） | 正确量角画角 |
| 平行四边形和梯形 | 平行与垂直，画垂线，平行四边形和梯形的特征 | 理解特征，正确画图 |
| 三角形 | 分类（按角/按边）、内角和(180°)、三边关系 | 掌握三角形性质 |
| 公顷和平方千米 | 大面积单位 | 建立观念 |

### 统计与概率
| 知识点 | 核心内容 |
|--------|----------|
| 条形统计图 | 单式和复式条形统计图，读取与分析 |

### 综合实践
- 数学广角——优化：沏茶问题、烙饼问题、田忌赛马
- 鸡兔同笼

## 五年级

### 数与代数
| 知识点 | 核心内容 | 课标要求 |
|--------|----------|----------|
| 小数乘法 | 小数乘整数、小数乘小数，积的近似数 | 掌握算理算法 |
| 小数除法 | 除数是整数/小数的除法，循环小数 | 正确计算 |
| 简易方程 | 用字母表示数，等式的性质，解方程 | 掌握方程思想 |
| 因数和倍数 | 因数、倍数、2/3/5的倍数特征，奇偶性 | 理解整除概念 |
| 质数和合数 | 100以内质数表，分解质因数 | 正确判断 |
| 分数的意义和性质（深入） | 分数与除法，真/假/带分数互化 | 深入理解分数 |
| 分数的加减法 | 异分母加减（通分），分数加减混合 | 掌握通分约分 |
| 最大公因数和最小公倍数 | 求法（列举/短除），约分和通分 | 熟练计算 |

### 图形与几何
| 知识点 | 核心内容 | 课标要求 |
|--------|----------|----------|
| 观察物体（三） | 根据三视图还原立体图形 | 发展空间观念 |
| 长方体和正方体 | 特征、展开图 | 理解特征 |
| 表面积 | 长方体/正方体表面积计算公式 | 掌握公式 |
| 体积 | 体积含义、体积单位、体积计算(V=abh, V=a³, V=sh) | 理解体积概念 |
| 容积 | 容积单位（L, mL），不规则物体体积 | 掌握换算 |

### 统计与概率
| 知识点 | 核心内容 |
|--------|----------|
| 折线统计图 | 单式和复式折线统计图，趋势分析 |
| 可能性 | 确定事件与不确定事件，可能性大小 |

### 综合实践
- 数学广角——找次品
- 打电话（优化问题）
- 探索图形（正方体涂色规律）

## 四大领域知识进阶脉络

### 数与代数
```
一年级: 20/100以内加减
  ↓
二年级: 表内乘除 → 万以内数
  ↓
三年级: 多位数乘除 → 分数/小数初步
  ↓
四年级: 大数运算 → 运算定律 → 小数
  ↓
五年级: 小数乘除 → 方程 → 分数深入
  ↓
六年级: 分数乘除 → 比和比例 → 百分数
```

### 图形与几何
```
1-2年级: 立体/平面图形认知、长度初步
  ↓
3-4年级: 周长 → 面积 → 角的度量 → 三角形/四边形性质
  ↓
5-6年级: 表面积 → 体积 → 圆 → 圆柱/圆锥
```

### 统计与概率
```
低段: 分类整理、简单统计表
  ↓
中段: 条形统计图、复式表格
  ↓
高段: 折线图、扇形图、可能性
```

## 关键概念衔接点（教学重难点过渡）

| 衔接 | 前概念 | 后概念 | 认知跨越 |
|------|--------|--------|----------|
| 二→三年级 | 表内乘除 | 多位数乘除 | 从口诀到笔算算法 |
| 三→四年级 | 初步分数 | 分数意义 | 从具体到抽象 |
| 三→四年级 | 周长 | 面积 | 从一维到二维 |
| 四→五年级 | 小数 | 小数乘除 | 从加减到乘除 |
| 四→五年级 | 算术 | 简易方程 | 从确定运算到符号代数 |
| 五→六年级 | 分数加减 | 分数乘除 | 从通分到约分乘除 |
| 五→六年级 | 面积 | 体积 | 从二维到三维 |
