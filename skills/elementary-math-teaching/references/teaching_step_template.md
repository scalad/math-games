# 教学步骤拆解模板

> 本模板参考内容策划团队的实际文档格式，将小学数学知识点拆解为结构化教学方案，同时标注游戏化映射点。

---

## 模板结构

每个知识点按以下模块拆解。`[游戏映射]` 标记为后续游戏设计提供直接输入。

---

### 一、课题信息

```
课题：（知识点名称，如"1~5的认识"）
课型：（新授课 / 练习课 / 复习课）
课时：（通常 1 课时）
年级：（对应年级及学期）
所属模块：（数与代数 / 图形与几何 / 统计与概率 / 综合实践）
前置知识：（2-3 个依赖的知识点）
后续关联：（本知识点支撑哪些后续内容）
```

---

### 二、教学目标

按三维目标撰写：

```
知识与技能：（学生能辨认/朗读/书写/计算/描述……）

过程与方法（核心素养）：（学生经历什么样的认知过程——如从实物观察到图示表征再到抽象符号。
                      标注所用教学法，如 CPA教学法、探究式学习等）

情感态度与价值观：（通过什么样的活动，感受数学与生活的联系，激发什么样的内在动力）
```

---

### 三、教学重难点

```
重点：（本课最需要学生掌握的核心内容）
难点：（学生认知上的跨越点或易混淆点）
突破策略：（用什么方法化解难点，如"采用 CPA 教学法搭建认知脚手架"）
```

---

### 四、教学方法

列出本课采用的主要教学法，说明各自服务于什么：

```
- （方法名）：（简要说明 + 服务于什么。如"情境引入法：创设贴合学生真实生活的情境，服务于情境代入与主动观察"）
- （方法名）：...
```

---

### 五、教学过程 [游戏映射]

教学过程按四环节组织，对应 CPA（具象 C → 意象 P → 抽象 A）认知链条。

#### 第一环节：创设情境，发现问题

```
（如为起始概念课，直接由生活化情境引入；如为后续课，可设计复习引入或冲突情境）

生活化情境：
（一个完整的、贴近学生生活的叙事场景）

学法设计：
（学生在这个环节怎么做——代入角色、观察画面、口头表述等）

指导点拨：
（教师的核心提问和引导语）

思维渗透：
（在本环节中渗透什么数学思想——有序观察、一一对应、分类、归纳等）

弹性生成预设：
（预设学生可能出现的典型困难 + 应对方案）

游戏映射：
（这个环节可以转化为什么游戏交互——如农场寻宝、点数配对等）
```

#### 第二环节：自主探究，建构新知

由 2-3 个递进活动组成，对应 CPA 三阶段：

```
活动1：（名称，对应 C — 直观体验）
- 学生活动：（具体操作描述）
- 分工设计：（不同层次学生的参与方式）
- 教师引导：（关键提问和巡视重点）
- 过程性评价：（即时反馈方式，如同桌互评、掌声鼓励）
- 思维渗透：
- 弹性生成预设：
- 游戏映射：

活动2：（名称，对应 P — 图示表征）
- 学生活动：
- 教师引导：
- 思维渗透：（如体会"1添上1是2"的递进关系）
- 弹性生成预设：
- 游戏映射：

活动3：（名称，对应 A — 抽象建模，可选）
- 学生活动：
- 教师引导：
- 思维渗透：
- 弹性生成预设：
- 游戏映射：
```

```

（将数学知识连接到真实生活和价值认同）

### 七、游戏化整体映射 [游戏映射]

```

## 使用说明

- 用户输入知识点名称 → 加载本模板 → 按模块顺序输出完整拆解
- 教学过程是核心，占整体篇幅的 60% 以上
- 参考示例：`references/concept_example.md`（分数的初步认识）
- CPA 教学法为本模板的默认认知框架
