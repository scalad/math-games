## 使用建议

### 作为 HTML 原型设计参考

1. **查找同类游戏**: 确定目标知识点后，在上方分类详情中找到对应分类，浏览该分类下的游戏列表
2. **参考玩法机制**: 每款游戏的 AI 详细玩法描述包含核心机制、策略技巧和教学建议
3. **跨年级借鉴**: 如果目标年级的游戏较少，可以查看相邻年级的同分类游戏，降低或提升复杂度
4. **多人模式灵感**: 标注"是"的游戏有竞争/合作元素，适合设计课堂互动原型

### 与12族系82种玩法的关系

- 本库的 225 款游戏偏向**在线互动型**（HTML/Web 原生），与已有 `game_mechanics_catalog.md` 的 82 种**课堂实体型**玩法互补
- HTML 原型设计时优先参考本库的数字化互动模式，教学设计时优先参考 82 种玩法的教育逻辑
- 两者结合使用：用本库找到"数字游戏长什么样"，用 82 玩法库找到"课堂活动怎么做"
