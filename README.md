# 🎮 小学数学互动游戏中心

基于《义务教育数学课程标准（2022年版）》设计的人教版配套数学游戏合集。

## ✨ 特性

- 🎯 **弹窗即玩** — 点击游戏卡片，全屏弹窗体验，ESC 一键关闭
- 🧩 **5 大玩法族系** — 角色·卡牌·推理·笔纸·竞速，覆盖不同思维模式
- 📊 **自适应练习** — 通关后自动匹配 5 档难度练习题（<40%→≥95%）
- 📱 **全设备支持** — 电脑平板流畅运行，触屏友好
- 📐 **可扩展** — 多知识点架构，随时新增模块

## 🗂️ 目录结构

```
math-games/
├── index.html                    # 统一主页（支持多知识点）
├── 教学方案/                      # 详细教案（7模块）
├── 游戏设计文档/                   # 每款游戏的GDD
├── 游戏原型/                      # 可玩HTML游戏（含自适应练习）
```
