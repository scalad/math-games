# 🎮 小学数学互动游戏中心

基于《义务教育数学课程标准（2022年版）》设计的人教版配套数学游戏合集。

## ✨ 特性

- 🎯 **弹窗即玩** — 点击游戏卡片，全屏弹窗体验，ESC 一键关闭
- 🧩 **6 大玩法族系** — 竞速·卡牌·运气·笔纸·推理·角色，覆盖不同思维模式
- 📊 **自适应练习** — 通关后自动匹配 5 档难度练习题（<40%→≥95%）
- 📱 **全设备支持** — 电脑平板流畅运行，触屏友好
- 📐 **可扩展** — 多知识点架构，随时新增模块
- 🧰 **Skill 工具包** — 内置教学步骤拆解 & 游戏设计师两大 AI Skill

## 🗂️ 目录结构

```
math-games/
├── index.html                    # 统一主页（支持多知识点）
├── skills/                       # AI Skill 工具包
│   ├── elementary-math-teaching/ # 教学步骤拆解 Skill
│   └── math-game-designer/      # 数学游戏设计师 Skill
├── 教学方案/                      # 详细教案 + 考试试题
├── 游戏设计文档/                   # 每款游戏的 GDD
└── 游戏原型/                      # 可玩 HTML 游戏（含自适应练习）
```

## 🚀 本地使用

直接用浏览器打开 `index.html` 即可运行，无需任何构建工具或依赖。

## ⚡ 新增知识点

在 `index.html` 中复制知识点模板 block，填入新游戏的路径和描述即可。
