# 游戏设计文档：数字记忆翻翻乐

> **知识点:** 1~5 的认识
> **适用年级:** 一年级上册
> **游戏机制类型:** B01 配对翻牌（卡牌族）
> **设计编号:** 02

---

## 一、核心概念

农场里来了一群数字小精灵，它们藏在卡片下面！每两张卡片是一对好朋友——一张是"物品图"（如3只小鸡🐤🐤🐤），另一张是"数字符号"（如"3"）。小朋友需要翻开卡片，找到匹配的一对。这不仅是记忆力的考验，更是在每一次翻开时强化"3个东西 = 数字3"的核心概念。三关递进：物品↔数字 → 点子图↔数字 → 数字↔数字的中文读法，层层深入！

---

## 二、教学目标映射

| 教学步骤 | 游戏中的体现 | 实现方式 |
|---------|------------|---------|
| 活动1：实物点数（C） | 第1关配对：物品图 ↔ 数字 | 翻开一张"4只兔子🐰"，需找到数字"4"的卡片 |
| 活动2：点子图图示（P） | 第2关配对：点子图 ↔ 数字 | 翻开一张"●●●●●"，需找到数字"5"的卡片 |
| 活动3：抽象符号（A） | 第3关配对：数字 ↔ 中文读法 | 翻开"3"，需找到"三"的卡片 |
| 练习：书写记忆 | 配对成功动画 | 配对成功时数字以田字格动画展示笔顺 |

---

## 三、核心玩法循环

```
开始 → 展示卡片背面（6/8/10张）
     → 玩家点击翻开第一张 → 观察内容
     → 玩家点击翻开第二张 → 判断是否配对
     →
       ├─ 配对成功：🎉 两张卡片亮起+保留翻开+得分
       └─ 配对失败：两张卡片翻回+扣减一次尝试机会
     → 所有卡片配对完成 → 过关
```

1. **观察:** 看到一堆背面朝上的卡片，需要记住之前翻开过的卡片位置。
2. **思考:** "刚才翻开的●●●在哪？它应该和数字3配对。我还记得那张数字3在哪里吗？"
3. **操作:** 依次点击两张卡片翻开。
4. **反馈:** 配对成功→两张卡片高亮闪烁+"太棒了！3只小鸡 = 数字3！"；失败→卡片短暂显示后翻回。
5. **进阶:** 卡片数量递增（6→8→10张），配对类型变化。

---

## 四、思维激发

**主要思维类型:** 模式识别 + 空间记忆

**如何激发思考:**
- 每次翻开一张卡片，玩家需要快速进行"**量→数**"的心理转换（看到3只小鸡→脑中浮现数字3），这是数感的核心训练
- 配对失败时系统会短暂展示两张卡片再翻回，玩家需要**记住位置**以便下次尝试——空间工作记忆训练
- 第3关引入中文数字"一、二、三、四、五"，建立**多种符号系统的对应关系**（阿拉伯数字↔中文数字）

## 五、视觉与交互设计

**视觉风格:** 童话卡片风格——像一本精美的童话书，卡片有金色边框，背面是星星月亮图案

**配色方案:**
- 主色：#7C4DFF（魔法紫）
- 辅色：#FFD54F（金色）
- 强调色：#00E676（配对成功绿）

**角色/元素:**
- 卡片：圆角方形，背面是闪烁的星星图案
- 物品图：可爱的动物/水果简笔画
- 数字符号：大号彩色数字，田字格背景
