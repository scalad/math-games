# 游戏设计文档：圆的记忆配对

> **知识点:** 圆的认识和画圆的方法
> **适用年级:** 六年级上册
> **游戏机制类型:** B01 配对翻牌（卡牌族）
> **设计编号:** 03

---

## 三、核心概念

桌面上铺着16张面朝下的卡片——8张是圆的概念术语/图示/符号，8张是它们的对应解释。翻两张，配对成功收走！比如翻开"半径 r"和"从圆心到圆上任意一点的线段"就是一对。"圆心 O"和"圆的中心点"是一对。还有特殊配对："d=2r"要和图形证据配对——翻开一张写着"d=2r"的卡和一张展示"直径=两条半径"的示意图。全部配对完成获"圆之记忆大师"称号！

## 二、教学目标映射

| 教学步骤 | 游戏中的体现 | 实现方式 |
|---------|------------|---------|

| 活动1：d=2r关系 | 特殊配对：公式↔图示 | 翻开"d=2r"和"直径由两条半径组成"的图示 |

| 练习：画圆方法 | 配对"画圆三步"↔"定圆心→定半径→旋转一周" | 步骤配对 |

## 三、核心玩法循环

```
开始 → 16张卡片面朝下 → 点击翻开第一张 →
  点击翻开第二张 →
  ├─ 配对成功 → 两张卡片翻转消失 + 得分 → 剩余卡片减少
  └─ 配对失败 → 两张卡片翻回去 → 记住位置
→ 重复直到所有卡片配对完成 → 结算得分
```

---

## 四、思维激发

**主要思维类型:** 记忆匹配 + 概念关联 + 归纳总结

**如何激发思考:**
- 非直接的文字匹配——需要理解概念才能判断配对（如"直径"的图示是一张穿过圆心的线段图）
- 干扰项设计：相近概念的卡片（如"半径"和"直径"的图示容易混淆）
- 配对时需要口头确认"为什么它们是一对"，强化概念内化

## 五、视觉与交互设计

**视觉风格:** 彩色卡通风——圆角卡片、鲜艳配色、大字体

**配色方案:**
- 主色：#3498db（蓝色卡片）
- 辅色：#e74c3c（红色卡片）
- 配对成功：#2ecc71（绿色闪光）

**卡片设计:** 术语卡片蓝底白字，图示卡片白底彩图，便于区分两类
