# 游戏设计文档：圆规大师

> **知识点:** 圆的认识和画圆的方法
> **适用年级:** 六年级上册
> **游戏机制类型:** D04 画图设计（操作族）
> **设计编号:** 01

---

## 一、核心概念

你获得了一把神奇的虚拟圆规！每关给出一个任务——画一个指定半径（或直径）的圆。你需要先确定圆心位置（点击画布），然后拖动设定半径，系统实时检测半径精确度。画得太小或太大都不行，必须与目标半径匹配。画出的圆越精准，得分越高！完成基础任务后，还有快速画圆挑战、盲画挑战等着你——让你深入理解"定点+定长"的画圆本质。

---

## 二、教学目标映射

| 教学步骤 | 游戏中的体现 | 实现方式 |
|---------|------------|---------|
| 活动2：圆规画圆 | 全部5关 | 虚拟圆规三步：定点→定长→画圆 |
| 圆规结构对应 | 界面提示 | 针尖=圆心O，两脚距离=半径r |
| d=2r 逆向计算 | 第3-4关 | 给定直径，玩家需心算半径再画 |
| 练习画圆 | 第5关 | 多种半径连续画圆 |

---

## 三、核心玩法循环

```
开始 → 阅读任务(画半径Xcm的圆) →
  点击画布定圆心 → 拖动确定半径(显示实时长度) →
  释放确认 → 系统检测误差 →
  ├─ 误差≤5% → "完美!" → 高分 → 下一关
  ├─ 误差≤15% → "不错!" → 中分 → 提示→重试或继续
  └─ 误差>15% → "再调整!" → 低分 → 重新画
→ 全部通关 → 结算
```

## 五、视觉与交互设计

**视觉风格:** 绘图工作台风格——白色画纸背景，木质边框，圆规工具图标
**配色方案:** 主色 #5C6BC0（蓝紫），辅色 #FFF（画纸），强调色 #FF7043

## 六、难度递进设计

| 关卡 | 数学难度 | 任务 |
|:----:|---------|------|
| 第1关 | 基础 | 画r=3cm的圆，有辅助网格 |
| 第2关 | 基础 | 画r=4.5cm的圆 |
| 第3关 | 进阶 | 画d=8cm的圆（需算r=4cm） |
| 第4关 | 进阶 | 画d=5cm的圆（需算r=2.5cm），无网格 |
| 第5关 | 挑战 | 连续画3个不同半径的圆 |
