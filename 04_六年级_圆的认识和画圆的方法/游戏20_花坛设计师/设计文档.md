# 游戏设计文档：花坛设计师

> **知识点:** 圆的认识和画圆的方法
> **适用年级:** 六年级上册
> **游戏机制类型:** F01 角色扮演（角色族）
> **设计编号:** 05

---

## 五、核心概念

你是一位城市规划师，负责在新建的社区公园中设计圆形花坛。每关有一个场景需求：比如"在公园中心广场设计一个半径5米的圆形喷泉花坛"、"在两个花坛之间设计一个直径8米的圆形草坪"。你需要在网格地图上：选择圆心位置→设定半径→画圆→验证尺寸。相邻花坛不能重叠，尺寸必须精确符合要求。完成所有花坛设计后，社区开放，居民们来到公园为你打分！

---

## 二、教学目标映射

| 教学步骤 | 游戏中的体现 | 实现方式 |
|---------|------------|---------|
| 情境引入：圆形花坛 | 第1关：设计单个圆形花坛 | 在网格地图上，点击定圆心，拖动定半径 |
| 活动2：画圆方法 | 操作过程：定点→定长→画圆 | 鼠标操作模拟画圆三步 |
| 练习：d=2r | 第3-4关：给定直径要求，需计算半径 | 需求写"直径6m的花坛"，玩家心算r=3m |
| 拓展：综合应用 | 第5关：多花坛布局，考虑间距不重叠 | 多个圆的位置规划 |

---

## 三、核心玩法循环

```
开始 → 阅读任务卡片(花坛规格要求) →
  在网格地图上点击选择圆心 → 拖动设定半径 →
  预览圆形 → 确认放置 →
  ├─ 尺寸正确 + 不重叠 → "✅完美!" → 花坛建成 → 下一任务
  └─ 尺寸错误或重叠 → 提示问题 → 调整重新放置
→ 所有花坛完成 → 社区开放动画 → 评分结算
```

---

## 四、思维激发

**主要思维类型:** 空间规划 + 创造性思维 + 综合应用

**如何激发思考:**
- 多花坛布局需考虑**空间关系**：两个圆不能重叠，同时要美观
- 从直径推半径的**逆向思维**持续训练
- "美观度"评分鼓励**创造性设计**——花坛排列成图案有额外加分

---

## 五、视觉与交互设计

**视觉风格:** 卡通城市风格——绿色草地网格地图，彩色花坛，可爱小建筑

**配色方案:**
- 主色：#27ae60（草地绿）
- 辅色：#3498db（蓝天/水池）
- 花坛色：#e74c3c, #f39c12, #9b59b6（多种花坛颜色）

## 六、难度递进设计

| 关卡 | 数学难度 | 任务要求 | 新元素 |
|:----:|---------|---------|--------|
| 第1关 | 基础 | 1个花坛，给定半径整数 | 操作引导 |
| 第2关 | 基础 | 2个花坛，给定各自半径 | 多花坛，不重叠 |
| 第3关 | 进阶 | 给定直径→需算半径→画圆 | 逆向计算 |
| 第4关 | 挑战 | 3个花坛，混合半径/直径要求 | 空间规划 |
| 第5关 | 拓展 | 自由设计：给定总面积，自行规划 | 创造性设计+尺寸约束 |
