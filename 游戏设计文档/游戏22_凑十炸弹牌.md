# 游戏设计文档：凑十炸弹牌

> **知识点:** 凑十法（20以内进位加法）
> **适用年级:** 二年级
> **游戏机制类型:** B08 炸弹牌（卡牌族）
> **设计编号:** 02

---

## 一、核心概念

魔法森林里有一副神秘的数字牌——大部分是凑十题卡，但其中藏着几张"炸弹牌"！玩家轮流翻牌：翻到题目卡就答题得分，答对分数累加；翻到炸弹牌——"嘭！"——前面所有分数归零！你可以选择在何时"存入"分数（保险），但存入后就不能继续翻牌了。是继续冒险翻牌还是见好就收？心跳加速的策略博弈！

---

## 二、教学目标映射

| 教学步骤 | 游戏中的体现 | 实现方式 |
|---------|------------|---------|
| 活动1 十格阵直观(C) | 翻到"验证卡"时弹出十格阵动画 | 附加奖励牌，展示可视化验证 |
| 活动2 算式分解(P) | 题目卡上印有凑十分解式填空 | 如"8+5=10+□"，填剩余数 |
| 凑十口诀 | 正确反馈融入口诀 | "8找2，拆5得3，10+3=13！" |
| 策略选择训练 | 核心机制：翻牌/存入的决策 | 每回合选择"继续"还是"存入保险" |

---

## 三、核心玩法循环

```
展示 12 张牌（9张题目卡 + 3张炸弹牌），牌面朝下
  → 分数显示器：当前分(可失去) / 保险分(安全)
  → 玩家选择："翻牌" 或 "存入保险"
    ├─ 翻牌 → 翻到题目卡（如"9+7=?"）
    │   ├─ 答对 → 当前分 +10 → 继续选择
    │   └─ 答错 → 当前分 -5 → 继续选择
    ├─ 翻到炸弹牌 → "嘭！"当前分清零！
    └─ 存入保险 → 当前分转入保险分(安全) → 回合结束
  → 12张牌全部翻开 → 结算总分 = 保险分
```

1. **观察:** 12张牌面朝下，已知已翻开内容，未知剩余炸弹位置
2. **思考:** "还剩8张牌，已经翻开2张炸弹，还有1张……风险多大？"
3. **操作:** 点击牌翻面，答对题后决定继续/存入
4. **反馈:** 炸弹→屏幕震动+音效提示；存入→金币飞入保险箱
5. **进阶:** 关1(8张1炸弹)→关2(10张2炸弹)→关3(12张3炸弹)→关4(14张3炸弹+限时)→关5(16张4炸弹+凑整题)

---

## 四、思维激发

**主要思维类型:** 策略规划 + 风险评估

**如何激发思考:**
- 概率直觉：炸弹还剩多少？该不该继续？——培养风险判断
- 策略权衡：存入时机决定最终得分，有最优策略吗？
- 数学与决策融合：不是纯靠运气——答对率越高，翻牌越值得冒险

---

## 五、视觉与交互设计

**视觉风格:** 魔法牌桌，深紫绒布底色，金色边框卡牌

**配色方案:**
- 主色：深紫 #4A148C
- 牌背：紫金星纹渐变
- 题目卡：奶油白底
- 炸弹卡：红黑条纹 💣
- 保险箱：金色 #FFD700

**角色/元素:**
- 卡牌阵列（3×4 或 4×4 网格）
- 当前分/保险分 双显示器
- "存入保险"按钮
- 炸弹爆炸粒子动画

## 六、难度递进设计

| 关卡 | 牌数 | 炸弹数 | 题目类型 | 特殊机制 |
|------|:----:|:-----:|----------|----------|
| 第1关 | 8张 | 1张 | 9+几凑十 | 无时限，熟悉规则 |
| 第2关 | 10张 | 2张 | 8+几/7+几凑十 | 增加保险存入提醒 |
| 第3关 | 12张 | 3张 | 混合凑十 | 存入后本轮不可继续翻 |
| 第4关 | 14张 | 3张 | 混合+凑整(38+?=40) | 每题限时8秒 |
| 第5关 | 16张 | 4张 | 全类型混合 | 不可存入！纯冒险模式 |
