# 游戏设计文档：日历谜题大挑战

> **知识点:** 年、月、日的认识
> **适用年级:** 三年级下册
> **游戏机制类型:** E02 密室逃脱（推理族）
> **设计编号:** 3

---

## 一、核心概念

玩家被"困"在一间时间主题密室里，必须在月历上找到线索、解答年月日的数学谜题来获取密码，逐道解锁才能逃出。每道锁对应一个知识子概念，谜题设计精巧——不是简单答题，而是需要观察日历图案、推理日期关系才能破解密码。4道锁全部打开即可"逃出密室"。

---

## 二、教学目标映射

| 教学步骤 | 游戏中的体现 | 实现方式 |
|---------|------------|---------|
| 子概念2: 大月小月 | 第1道锁："密码是大月月份数×2" | 玩家需知道大月有7个→密码=14 |
| 子概念3: 平年闰年 | 第2道锁：日历显示2024年2月 | 观察日历有29天→判断闰年→密码=366 |
| 子概念4: 24时计时法 | 第3道锁：墙上的钟指向"3点" | 需转换为15→与另一线索组合得密码 |
| 子概念5: 日历阅读 | 第4道锁：找出月历中"第2个星期五" | 找到日期数作为密码一部分 |

---

## 三、核心玩法循环

```
进入密室 → 观察场景中的线索(日历、时钟、便签)
  → 第1道锁出现 → 解题(大月数量=? ) → 输入密码
  → 正确 → 锁🔓打开！进入下个区域 → 第2道锁...
  → 错误 → 提示"不对哦，数数一年有几个大月？" → 重试不减分
  → 第4道锁打开 → 🎉逃出密室！
```

1. **观察:** 场景中散布日历、钟面、线索便签
2. **思考:** 从多个线索中提取信息，组合成密码
3. **操作:** 点击线索查看，在密码盘输入答案
4. **反馈:** 错误给启发式提示（不直接给答案），正确播放锁打开动画
5. **进阶:** 每道锁的谜题逐级变难，从单知识点到多知识点组合

---

## 四、思维激发

**主要思维类型:** 逻辑推理、演绎推理

**如何激发思考:**
- 线索不是直接问题，需要玩家自己"发现"问题
- 错误提示引导玩家换角度思考
- 第4道锁需要综合前3道锁的知识

## 五、视觉与交互设计

**视觉风格:** 神秘解谜风格，暗色调配暖色灯光

**配色方案:**
- 主色：#2C3E50（深蓝灰）
- 辅色：#F39C12（金色光）
- 强调色：#E74C3C（锁/警示红）

**角色/元素:**
- 场景：带日历的墙壁、密码锁、飘浮的线索纸片
- 密码盘：数字转盘或输入框

## 六、难度递进设计

| 关卡/阶段 | 数学难度 | 线索数量 | 密码复杂度 |
|----------|---------|---------|----------|
| 锁1：月份之锁 | 基础 | 1个线索 | 直接计算 |
| 锁2：闰年之锁 | 进阶 | 2个线索 | 观察+判断 |
| 锁3：时钟之锁 | 进阶 | 2个线索 | 转换+运算 |
| 锁4：终极之锁 | 综合 | 3个线索 | 多步推理 |

## 八、游戏平衡数据

- 每道锁允许无限重试
- 提示系统：第1次错误→模糊提示；第2次错误→明确提示；第3次→显示解法
- 过关条件：4道锁全部解开
- 评分：总尝试次数越少→星级越高
