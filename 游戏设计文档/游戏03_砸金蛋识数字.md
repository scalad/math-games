# 游戏设计文档：砸金蛋识数字

> **知识点:** 1~5 的认识
> **适用年级:** 一年级上册
> **游戏机制类型:** I03 砸金蛋（运气族）
> **设计编号:** 03

---

## 一、核心概念

农场庆祝丰收节，摆出了5颗闪闪发光的金蛋！每颗金蛋里藏着一个与数字相关的题目——可能是"哪个是数字3？""图上有几个苹果？""点子图表示几？"小朋友选择一颗金蛋敲开，回答里面的题目，答对就能获得金蛋里的惊喜奖励（⭐、🌸、🎀）！5颗金蛋对应5道题，全部答对解锁"丰收小达人"称号。随机出题让每次游戏体验都不同！

---

## 二、教学目标映射

| 教学步骤 | 游戏中的体现 | 实现方式 |
|---------|------------|---------|
| 活动1：数字认读（A） | 数字识别题："哪个是数字4？" | 展示4个选项卡片，选出正确数字 |
| 活动2：基数含义（C→P） | 数量判断题："图上有几个苹果？" | 展示实物图，选择正确数量/数字 |
| 练习1：看数圈物 | 点数题："选出5颗星星" | 多个物品群，选出数量=目标数字的那组 |
| 总结：数感综合 | 混合随机题 | 金蛋随机出不同类型的题目 |

---

## 三、核心玩法循环

```
开始 → 展示5颗金光闪闪的金蛋
     → 玩家选择一颗金蛋 → 敲开动画 💥
     → 显示题目（随机类型）
     → 玩家选择/输入答案
     →
       ├─ 正确：🎉 "太棒了！" + 奖励飞出 + ⭐+1
       └─ 错误：💡 显示正确答案 + 提示
     → 金蛋被标记为已开启
     → 选择下一颗金蛋
     → 5颗金蛋全部开启 → 结算奖励
```

1. **观察:** 5颗金蛋一字排开，每颗都闪闪发光，让人想敲开看看。
2. **思考:** 题目出现后需要判断——是选数字、数数量还是认数字？
3. **操作:** 点击金蛋→敲开→选择题选答案/填空题输入数字。
4. **反馈:** 正确→蛋壳碎裂飞出彩带+奖励物品；错误→金蛋变成银色+"加油，再来！"
5. **进阶:** 5道题的难度逐渐提升（从数字识别→数量判断→混合推理）。

---

## 四、思维激发

**主要思维类型:** 快速判断 + 综合应用

**如何激发思考:**
- 题目类型随机混合（数字识别/数量判断/点子图），玩家需要在不同思维模式间切换
- 第4~5题引入"反向思维"：给出数字，要求从多个物品图中选出数量匹配的那组——逆向的数物对应
- 错误反馈中附带解题思路而非直接给答案："数一数图上有几个？手指点一下试试！"

---

## 五、视觉与交互设计

**视觉风格:** 丰收庆典风格——金色调为主，红丝带装饰，节日气氛浓厚

**配色方案:**
- 主色：#FFD700（金色）
- 辅色：#DC143C（庆典红）
- 强调色：#FFFFFF（亮白）

**角色/元素:**
- 金蛋：5颗大小不一的金色蛋，带有闪光粒子效果
- 奖励物品：⭐星星、🌸花朵、🎀蝴蝶结、🍬糖果、🏅奖牌
- 小锤子：鼠标/手指变成小锤子形状

## 六、难度递进设计

| 金蛋顺序 | 数学难度 | 题型 | 示例 |
|---------|---------|------|------|
| 第1颗 | 数字识别 | 选择数字 | "哪个是数字3？"[1][2][3][4] |
| 第2颗 | 简单点数 | 数物选数 | "图上有几个苹果？🍎🍎🍎" |
| 第3颗 | 点子图识数 | 点子→数字 | "●●●● 表示几？" |
| 第4颗 | 逆向：数字→数量 | 看图选组 | "哪组是5个？"[🐤×3][🐤×5][🐤×2][🐤×4] |
| 第5颗 | 综合：中文数字 | 混合题型 | "三" 等于哪个数字？ |
