# 公顷和平方千米 —— 四年级数学教学方案

> 基于《义务教育数学课程标准（2022年版）》四年级「图形与几何」领域。完整7模块结构化教学拆解 + 游戏设计 + 考试试题。

---

## 1. 知识点定位

```
- 所属年级：四年级上册
- 所属模块：图形与几何
- 前置知识：
  1. 面积的含义（三年级：物体表面或封闭图形的大小叫面积）
  2. 常用面积单位 cm²、dm²、m²（三年级：1cm²、1dm²、1m²的实际大小）
  3. 长方形和正方形面积公式（三年级：S = ab，S = a²）
  4. 长度单位 mm、cm、dm、m、km 及其进率（三年级：1km = 1000m）
- 后续关联：
  1. 五年级：体积和体积单位（面积→体积的维度跨越）
  2. 五年级：长方体正方体表面积计算
  3. 六年级：圆的面积（以 m² 和更大面积单位为背景）
  4. 初中地理：国土面积、行政区划面积等实际数据
- 教学目标：能正确建立1公顷和1平方千米的面积观念，掌握公顷、平方千米与平方米之间的进率及换算方法，能根据实际情况选择合适的面积单位。
```

## 2. 情境导入 [游戏映射]

### 情境一：学校操场有多大？

- **描述**：体育课上，小明和同学们在操场上跑步。老师问："我们学校的操场面积大约是5000平方米。你们能用学过的面积单位来描述它吗？"小明说："5000平方米，就是5000个1平方米的正方形拼在一起，太大了吧！"老师笑了："那有没有更大的面积单位，能让数字变小一点呢？"
- **引导问题**：
  1. 5000平方米用"平方米"做单位，数字大不大？方便吗？
  2. 你还能想到哪些"用平方米表示数字会很大"的地方？（学校、公园、城市……）
  3. 如果有一个比平方米更大的面积单位，你希望它有多大？
- **数学本质**：产生认知冲突——旧单位（m²）在描述大面积时数字过大，产生引入更大面积单位的必要性。
- **游戏映射**：可用"单位放大镜"道具——点击放大镜，面积单位从 cm² → dm² → m² → 公顷 → km² 逐级放大，展示同一块地用不同单位表示时数字的变化。

### 情境二：城市探秘——我们生活的城市有多大？

- **描述**：小红看城市宣传册，上面写着："我们城市的总面积约1500平方千米。"小红疑惑了："平方千米是什么？比平方米大多少？我们学校操场有几个平方千米？"妈妈说："你先想想，1千米有多长？1平方千米大概相当于多少个学校操场？"
- **引导问题**：
  1. "平方千米"这个名字让你想到了什么？（长度单位"千米"和面积单位"平方"的结合）
  2. 边长为1千米的正方形，面积是多少？这个面积大吗？
  3. 为什么不用"平方米"来描述城市面积？
- **数学本质**：平方千米是边长为1km的正方形的面积，是"千米"的平方。区别于之前学的 m²（米×米），引出"大面积单位 = 大长度单位的平方"这一模式。
- **游戏映射**：地图缩放互动——玩家在地图上框选区域，系统自动用不同单位显示面积，感受单位切换的便利性。

### 情境三：天安门广场和鸟巢

> （可选，用于引入公顷）

- **描述**：展示天安门广场图片。"天安门广场是世界上最大的城市广场，面积约44万平方米。新闻里常常说它的面积是'44公顷'。公顷又是什么？44公顷和44万平方米是一回事吗？"
- **引导问题**：
  1. 你能发现"公顷"这个名字的规律吗？（"公"=公制，"顷"=古代面积单位，1顷=100亩）
  2. 44万平方米 = 44公顷，猜猜1公顷等于多少平方米？
- **数学本质**：公顷的定义——边长为100米的正方形面积，1公顷 = 10000平方米。是平方米和平方千米之间的"桥梁单位"。
- **游戏映射**：地标解锁系统——玩家收集各地标面积数据（天安门44公顷、故宫72公顷、颐和园290公顷），用公顷和平方千米两种方式表达。

## 3. 核心概念拆解 [游戏映射]

### 子概念1：认识公顷（1 ha = 10000 m²）

- **定义**：边长100米的正方形的面积是1公顷。1公顷 = 10000平方米。
- **本质理解**：公顷就是"百米平方"——把100米当边长围出的正方形的面积。
- **关键特征**：
  1. 1公顷 = 100m × 100m = 10000m²
  2. 公顷的符号是"ha"或"hm²"（平方百米）
  3. 用于描述中等偏大的面积：公园、校园、广场、农田
- **正例**：
  1. 一个标准足球场面积约7140m² ≈ 0.7公顷 ✓
  2. 学校占地面积约2公顷（即20000m²）✓
- **反例**：
  1. 教室面积约50m² = 0.005公顷——用公顷描述教室太小，不方便 ✗
  2. 一个城市面积1500公顷——用公顷描述城市数字太大，不如用平方千米 ✗
- **教学提示**：
  - 核心操作：带学生在操场上实际走出100m×100m的正方形（可用步测），建立1公顷的直观感受
  - 关键追问："为什么边长选100米？"→ 因为100×100=10000，是一个整齐的数字
  - 预期难点：10000这个进率不是100（不像长度单位那样十进制），要特别强调
- **游戏映射**：地图网格模式——屏幕上显示100m×100m的网格（1公顷），玩家在网格上建造建筑，每个建筑有面积，计算占用了几公顷。

### 子概念2：认识平方千米（1 km² = 1000000 m² = 100 ha）

- **定义**：边长1千米（1000米）的正方形的面积是1平方千米。1平方千米 = 1000000平方米 = 100公顷。
- **本质理解**：平方千米就是"千米平方"——最大的常用面积单位，用来描述城市、省份、国家等超大区域。
- **关键特征**：
  1. 1 km² = 1000m × 1000m = 1,000,000 m²
  2. 1 km² = 100 ha（因为1km² = 100个100m×100m的正方形）
  3. 符号是 km²，读作"平方千米"
- **正例**：
  1. 中国台湾面积约3.6万平方千米 ✓
  2. 北京市面积约1.64万平方千米 ✓
  3. 一个大型湖泊面积约300平方千米 ✓
- **反例**：
  1. 教室面积50m² = 0.00005 km²——数字太小，不实用 ✗
  2. 地球表面积5.1亿km²——超出了小学阶段 km² 的常规使用范围 ✗
- **教学提示**：
  - 核心操作：用地图软件展示1km²的范围，让学生看到"从家到学校大概1km，这个距离围成的正方形就是1km²"
  - 关键追问："1平方千米等于多少公顷？为什么？"→ 100m×10排 = 1km，10×10=100个公顷
  - 画图法：画一个10×10的方格（每格=1公顷），整个大方格=1km²
- **游戏映射**：城市建造沙盒——玩家在km²级别的地图上规划城市，放大后进入公顷级别放置建筑。

### 子概念3：公顷和平方千米的换算关系

- **定义**：平方千米和公顷之间的换算是基于它们各自与平方米的关系推导的。
- **本质理解**：1 km² : 1 ha = 100 : 1。把1平方千米想象成10×10的"公顷棋盘"。
- **关键特征**：
  1. 大→小：1 km² = 100 ha = 1,000,000 m²
  2. 小→大：1 ha = 0.01 km²；1 m² = 0.0001 ha = 0.000001 km²
  3. 记忆口诀："平方千米转公顷，小数点右移两位"
- **正例**：
  1. 3 km² = 3 × 100 = 300 ha ✓
  2. 500 ha = 500 ÷ 100 = 5 km² ✓
- **反例**：
  1. 1 km² = 1000 ha ✗（混淆了长度进率1000和面积进率100）
  2. 1 ha = 100 km² ✗（方向搞反了）
- **教学提示**：
  - 用"棋盘图"可视化：一个10×10的棋盘，每格=1公顷，整盘=1km²
  - 让学生动手画并数格子
- **游戏映射**："单位连连看"——给出不同单位的面积数值，玩家用线连接等值的两个。

### 子概念4：选择合适的面积单位

- **定义**：根据描述对象的大小，在 cm²、dm²、m²、公顷、km² 中选择最合适的单位，使得数值在易读范围（通常0.1~1000之间）。
- **本质理解**：选单位 = 让数字"刚刚好"——不太大也不太小，读起来舒服。
- **关键特征**：
  1. cm²：指甲盖、邮票（约1~100 cm²）
  2. dm²：手掌、书本封面（约1~10 dm²）
  3. m²：房间、黑板（约1~100 m²）
  4. 公顷：校园、公园、农场（约1~1000 ha）
  5. km²：城市、省份、国家（约1~10000 km²）
- **正例**：
  1. 课桌面 → dm² ✓
  2. 学校操场 → m² 或 公顷 ✓
  3. 中国香港面积 → km² ✓
- **反例**：
  1. 橡皮擦面积用 m² ✗
  2. 一个省的面积用公顷 ✗（数字会非常大）
- **教学提示**：
  - 做一个"面积单位阶梯"：从 cm² → dm² → m² → ha → km²，让学生贴上对应的实物图片
- **游戏映射**："单位侦探"——系统给出描述对象和数字，玩家选择正确的单位。选错时给出形象的对比提示。

### 子概念5：面积单位换算与计算应用

- **定义**：在实际问题中进行不同面积单位之间的换算，并用面积公式解决包含大面积单位的实际问题。
- **本质理解**：统一单位 → 计算 → 按需要换算回合适的单位。
- **关键特征**：
  1. 换算时必须先统一单位
  2. 利用面积公式（长方形/正方形）先算出面积
  3. 结果按习惯选择最合适的单位表达
- **正例**：
  1. 一块长方形农田长500m、宽200m，面积 = 500×200 = 100,000m² = 10公顷 ✓
  2. 一个正方形公园边长2km，面积 = 2×2 = 4 km² = 400公顷 ✓
- **反例**：
  1. 长用米、宽用千米直接乘 ✗（必须先统一单位）
  2. 300m × 400m = 120,000m，写成120,000m² ✓ 但学生漏写"²" ✗
- **教学提示**：
  - 强调"单位不统一先统一"
  - 强调面积单位的"平方"符号不能丢
- **游戏映射**："农场大亨"——玩家拥有农田，需要计算面积、选择单位、比较不同地块的大小。

## 4. 例题与练习 [游戏映射]

### A层 — 基础巩固

#### 例1：基本换算
- **题目**：3公顷 = ______ 平方米；50000平方米 = ______ 公顷
- **解题过程**：
  - 3公顷 = 3 × 10000 = 30000平方米
  - 50000平方米 = 50000 ÷ 10000 = 5公顷
- **设计意图**：建立公顷↔平方米的直接换算技能，巩固进率10000。

#### 例2：高级换算
- **题目**：2平方千米 = ______ 公顷 = ______ 平方米
- **解题过程**：
  - 2平方千米 = 2 × 100 = 200公顷
  - 200公顷 = 200 × 10000 = 2,000,000平方米
- **设计意图**：练习 km²→ha→m² 的链式换算。

#### 例3：单位选择
- **题目**：在括号里填上合适的单位——(1) 一个足球场面积约7000( )；(2) 中国澳门面积约33( )
- **解题过程**：
  - (1) 足球场：7000这个数字如果填m²，约等于0.7公顷，合理→ 平方米 ✓
  - (2) 中国澳门：33后面如果填km²，33平方千米合理；如果填公顷=3300公顷=33km²，数字还行，但习惯用km²描述地区面积→ 平方千米 ✓
- **设计意图**：培养根据数值大小和实际情境选择合适单位的判断力。

### B层 — 综合应用

#### 变式1：长方形面积 + 单位换算
- **题目**：一块长方形的稻田，长400米，宽250米。这块稻田的面积是多少平方米？合多少公顷？
- **解题过程**：
  - 面积 = 400 × 250 = 100000（平方米）= 100000 m²
  - 100000 ÷ 10000 = 10（公顷）
  - 答：100000平方米，合10公顷。
- **变式意图**：将面积公式计算与单位换算结合，体现知识的综合运用。

#### 变式2：逆向推算
- **题目**：一个正方形公园的面积是64公顷。它的边长是多少米？
- **解题过程**：
  - 64公顷 = 640000平方米
  - 正方形边长 = √640000 = 800（米）
  - 答：边长是800米。
- **变式意图**：从面积反推边长，涉及大数开方——可以用"800×800=640000"验证，不需要正式开方。

#### 变式3：多步换算比较
- **题目**：A公园面积是3平方千米，B公园面积是280公顷。哪个公园更大？大多少公顷？
- **解题过程**：
  - A公园：3 km² = 300公顷
  - B公园：280公顷
  - 300 - 280 = 20（公顷）
  - 答：A公园更大，大20公顷。
- **变式意图**：先统一单位再比较，训练"不同单位不能直接比较"的意识。

**B层练习题**（共4题）：
1. 一个长方形果园长500米、宽160米，面积是多少公顷？
2. 一个正方形广场周长是1200米，面积是多少公顷？
3. 比较大小：2.5 km² ○ 248 公顷（填 > < =）
4. 某开发区面积是15平方千米，合多少公顷？如果每公顷建一栋楼，最多建多少栋？

> **游戏映射**：B层 → "农场大亨"经营模式——玩家管理不同形状的农田，需要计算面积、换算单位、比较产量，答对才能收获。

### C层 — 拓展提高

#### 变式：综合分析
- **题目**：某城市老城区面积12平方千米，新城区比老城区大800公顷。这个城市的总面积是多少平方千米？
- **解题过程**：
  - 800公顷 = 800 ÷ 100 = 8平方千米
  - 新城区 = 12 + 8 = 20平方千米
  - 总面积 = 12 + 20 = 32平方千米
  - 答：总面积32平方千米。
- **变式意图**：同一个问题中混合使用平方千米和公顷，需要灵活换算。同时训练多步推理。

**C层练习题**（共2题，选做）：
1. 一个长方形农场，长3千米、宽2千米。将其中的500公顷划为种植区，剩下的面积是多少平方千米？
2. 探究题：如果你家有1公顷的土地，你想怎样规划？（画出示意图，标注各区域面积，用m²和公顷两种方式表达）

> **游戏映射**：C层 → "市长挑战"模式——作为城市规划师，需要处理不同单位的面积数据，做出最优规划方案。限时挑战，考验综合能力。

## 5. 常见错误

### 错误1：进率记忆混乱

- **典型表现**：学生写出"1公顷 = 1000平方米"或"1平方千米 = 10000公顷"
- **认知根源**：受长度单位进率（1km=1000m）影响，错误地把面积进率也当成1000；或者把 ha→km² 的进率100记成10000（与 ha→m² 的进率混淆）。
- **纠正策略**：
  1. 用画图法还原：在纸上画100m×100m的正方形，一格一格数（虽然不实际数完），体会100×100=10000
  2. 对比记忆：长度进率是10、100、1000（线性）；面积进率是100、10000、1000000（平方）
- **预防措施**：
  - 教学时反复强调"面积是二维的！进率是长度进率的平方！"
  - 每次换算都先让学生说出"边长的进率是多少？面积的进率就是它的平方"

### 错误2：单位选择不当

- **典型表现**：教室面积填"公顷"，或一个省的面积填"公顷"
- **认知根源**：缺乏对各个面积单位"实际有多大"的直观感受——学生记的是数字和换算公式，但没有建立"体感"。
- **纠正策略**：
  1. 制作"面积单位参照物卡片"：手掌≈1dm²，教室地面≈50m²，操场≈0.5公顷，小区≈5公顷，城区≈50km²
  2. 排序游戏：给出一组面积+单位，让学生按实际大小排序
- **预防措施**：
  - 教学每个单位时必须让学生"比划"出它的大小，不能只记忆定义
  - 课堂设置"单位选择站队"活动：老师说一个场景，学生举相应的单位卡片

### 错误3：换算方向搞反

- **典型表现**：把"公顷→平方米"除以10000而不是乘以10000；5公顷=0.0005平方米
- **认知根源**：对"大单位→小单位数字变大，小单位→大单位数字变小"这一基本规律掌握不牢，在涉及不熟悉的进率时容易混乱。
- **纠正策略**：
  1. "大化小，乘进率；小化大，除以进率"——用熟悉的例子（1m=100cm）类比
  2. 直觉检查法："公顷比平方米大，所以同样的面积用公顷表示数字应该更小"
- **预防措施**：
  - 教学生养成"换完回头想"的习惯——换算完成后用目测法判断结果是否合理

### 错误4：面积计算后忘记加"²"

- **典型表现**：400m × 250m = 100000m（漏写²）
- **认知根源**：学生对面积的"二维性"理解不深，把面积计算等同于普通的数字乘法，忽略了单位的平方含义。
- **纠正策略**：
  1. 口诀："m×m=m²，单位也要乘起来"
  2. 展示对比：400m（长度，一条线）vs 100000m²（面积，一大片）
- **预防措施**：
  - 每次列算式都写出完整单位：400m × 250m = 100000 m²
  - 让学生养成"写完数字看单位"的习惯

## 6. 游戏化建议 [游戏映射]

### 核心玩法

- **游戏类型**：模拟经营 + 策略规划（沙盒建造）
- **核心循环**：获得土地 → 测量/计算面积 → 选择合适单位表达 → 规划建设 → 获得评分 → 解锁更大地图
- **数学操作映射**：
  - "认识公顷/平方千米" → 在网格地图上选取100m×100m或1km×1km区域
  - "单位换算" → 切换显示模式（㎡视图 / 公顷视图 / km²视图）
  - "面积计算" → 使用公式计算不规则组合地块的面积
  - "单位选择" → 为不同建筑自动推荐最合适的面积单位

### 关卡设计

| 关卡 | 对应子概念 | 场景 | 核心任务 | 通过条件 |
|------|-----------|------|----------|----------|
| 第1关 | 认识公顷 | 校园规划师 | 在100m×100m网格上放置教学楼、操场，计算各占几公顷 | 正确计算3个建筑的公顷数 |
| 第2关 | 认识平方千米 | 城市探险家 | 在城市地图上框选1km²范围，观察其中包含多少个100m×100m网格 | 正确回答"1km²=100ha" |
| 第3关 | 换算关系 | 单位魔法师 | 用"单位转换器"道具，将给出的面积在m²/ha/km²之间切换 | 完成10次换算，正确率≥90% |

### 反馈机制

- **正确反馈**："完美！这块地的面积是5公顷 = 50000平方米！" + 地块变绿 + 金币动画
- **错误反馈**：
  - 单位选错："这个单位太大了！一个房间用'平方千米'就像用'吨'来称一只蚂蚁。"（幽默类比提示）
  - 换算错误："想想看，1公顷是边长100米的正方形。5个这样的正方形并排……"（视觉化启发）
  - 面积计算错："长500米，宽200米，500×200=？再检查一下零的个数！"
- **进度可视化**：
  - 地图逐步解锁（从校园→公园→城市→国家）
  - 成就徽章系统：🟢"公顷达人"、🔵"换算高手"、🟡"单位侦探"、🔴"城市规划大师"
  - 每个关卡3星评价（速度分+正确分）

### 故事包装

- **主题**：「神奇放大镜 —— 从操场到世界」
- **情境**：玩家获得一把"神奇放大镜"，可以缩放视角——在"平方米视角"看到教室和操场，在"公顷视角"看到学校和公园，在"平方千米视角"看到城市和山川。玩家需要利用这把放大镜完成各种测绘任务，从"初级测绘员"晋升到"首席城市规划师"。

  - 序章：获得神奇放大镜，学习基础缩放（引入m²/ha/km²层级）
  - 第一章「校园测绘员」：用公顷规划校园（2-3个任务）
  - 第二章「公园设计师」：公顷与平方米混合换算（3-4个任务）

## 7. 关联拓展

### 知识链

```
回顾（三年级）                         本课（四年级）                    展望（五~六年级）
面积的含义                          公顷和平方千米                    体积单位
cm²/dm²/m²                        1ha=10000m²                     m³/dm³/cm³
S=ab, S=a²                        1km²=100ha=1000000m²           表面积计算
长度单位及进率                      大面积单位的换算与选择              圆的面积、扇形面积
```

**关键认知衔接**：从三年级"一维长度→二维面积"跨越后，四年级在本课实现"小面积→大面积"的量级扩展。五年级将再次跨越"二维面积→三维体积"。本课的核心价值在于让学生理解：**同样的"平方"逻辑，换不同的"边长单位"就能描述不同尺度的面积**。这为后续体积单位的学习建立了"换单位 = 换边长"的思维范式。

### 生活应用

1. **阅读新闻数据**：看到"某森林大火烧毁了800公顷林地"时，能理解灾难的规模；看到"某城市建成区面积达450平方千米"时，能判断这个城市的大小。
2. **旅行中的地理感知**：看地图时理解比例尺和面积的关系，比如比较两个湖泊的大小——"太湖面积约2400km²，西湖面积约6.4km²"。
3. **家庭决策参与**：和父母看房时，能理解"小区占地5公顷""这套房子120m²"意味着什么，用数学帮助家庭理解购房信息。

### 跨学科联系

- **科学**：自然保护区面积（如"卧龙自然保护区面积约2000平方千米"）
- **地理**：国土面积排名（俄罗斯1710万km²、加拿大998万km²、中国约960万km²）
- **体育**：标准足球场面积 ≈ 7140m² ≈ 0.714公顷，世界杯决赛场地规格
