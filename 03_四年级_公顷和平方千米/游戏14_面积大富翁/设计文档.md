# 游戏设计文档：面积大富翁

> **知识点:** 公顷和平方千米
> **适用年级:** 四年级
> **游戏机制类型:** C01 大富翁（棋盘族）
> **设计编号:** 4

---

## 一、核心概念

掷骰子走格子，每个格子触发不同的"面积任务"——可能是换算题、计算题、选择题或幸运事件。玩家用虚拟金币购买地块，地块面积以不同单位标注。最终拥有最大"总面积"（统一换算后比较）的玩家获胜。把枯燥的换算练习变成刺激的地产竞争。

---

## 二、教学目标映射

| 教学步骤 | 游戏中的体现 | 实现方式 |
|---------|------------|---------|
| 子概念1：认识公顷 | 棋盘上"公顷地块"格子 | 走到时需回答1ha=?m² |
| 子概念2：认识平方千米 | "平方千米地块"格子 | 购买需正确换算 |
| 子概念3：换算关系 | 核心任务卡 | 抽卡完成换算 |
| 子概念4：选择合适的单位 | "单位选择"事件格 | 选错单位扣金币 |
| 子概念5：计算应用 | "面积计算"挑战格 | 给出长宽算面积 |

---

## 三、核心玩法循环

```
掷骰子 → 前进N步 → 落在格子上 →
查看格子类型：
  ├─ 📐 面积题格 → 答对得金币/买地权
  ├─ 🎁 幸运格 → 随机奖励
  ├─ 💰 地产格 → 用金币购买（需正确换算）
  └─ ⚡ 事件格 → 特殊效果
→ 轮流进行 → N轮后结算 → 比较总资产和总面积
```

---

## 四、思维激发

**主要思维类型:** 策略规划 + 估算判断

**如何激发思考:**
- 购买地块时需判断"2公顷 vs 18000平方米 哪个更大？"
- 策略选择：是用金币买更多小地块，还是攒钱买一大块地？
- 事件格："你的地被征用了一半（从4公顷变为？平方米）"——需要现场换算

## 五、视觉与交互设计

**视觉风格:** 经典大富翁棋盘风，色彩明快的卡通城市地图

**配色方案:**
- 棋盘底色：米色 #FFF8E7
- 地产格：金色 #FFD700
- 题目格：蓝色 #4A90D9
- 幸运格：绿色 #52C41A

**角色/元素:** 棋子（角色头像）、骰子、地契卡、金币图标、任务卡

## 六、难度递进设计

| 阶段 | 格子配比 | 题目难度 | 特殊规则 |
|------|---------|---------|---------|
| 开局 | 50%地产 + 30%题目 | A层换算 | 基础买卖 |
| 中期 | 30%地产 + 50%题目 | B层计算 | 地产可交易 |
| 后期 | 20%地产 + 60%题目 | C层综合 | 面积税收 |

## 七、HTML原型范围

**原型包含:**
- [x] 16格环形棋盘
- [x] 掷骰子动画
- [x] 3种格子系统（题目/地产/幸运）
- [x] 金币和地产资产追踪
- [x] 人机对战（简单AI）
- [x] 题目弹窗

**原型暂不包含:**
- [ ] 多人联网
- [ ] 地产交易系统

**预计原型复杂度:** 高

---

## 八、游戏平衡数据

- 初始金币：500
- 答对奖励：+100金币
- 答错惩罚：-50金币
- 地产价格：100~500金币
- 游戏轮数：20轮
- 胜利条件：终局总资产最高
