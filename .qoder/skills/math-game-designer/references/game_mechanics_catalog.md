# 小学数学游戏素材库 — 玩法机制总表

> 一张表穷举所有数学教育游戏玩法，作为游戏设计的**素材字典**直接查用。

---

## 使用说明

| 列名 | 含义 |
|------|------|
| **编号** | 唯一编号，方便引用 |
| **玩法名称** | 通用名称（附英文/别名） |
| **分类** | 所属族系 |
| **核心机制** | 一句话说清怎么玩 |
| **经典变体** | 常见的变化玩法 |
| **适用知识点** | 最适合对接的数学内容 |
| **适用年级** | 1~6年级范围 |
| **所需道具** | 实施需要的材料 |
| **参与人数** | 适合多少人 |
| **节奏** | 快/中/慢 |
