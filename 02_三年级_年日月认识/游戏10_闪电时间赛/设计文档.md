# 游戏设计文档：闪电时间赛

> **知识点:** 年、月、日的认识
> **适用年级:** 三年级下册
> **游戏机制类型:** A02 限时挑战（竞速族）
> **设计编号:** 5

---

## 一、核心概念

30秒内尽可能多地正确回答年月日判断题和选择题！题目快速弹出，玩家必须在倒计时结束前点击正确答案。例如屏幕上闪过"2024年2月有( )天"，选项[28][29]——选29正确，选28错误。连击越多速度越快，题目越来越难。这是一个训练快速反应和知识熟练度的速度游戏。

---

## 二、教学目标映射

| 教学步骤 | 游戏中的体现 | 实现方式 |
|---------|------------|---------|
| 子概念2: 大月小月 | "7月有多少天？[30]/[31]" | 快速判断大月31天 |
| 子概念3: 平年闰年 | "2020年是(平年/闰年)" | 口算÷4判断 |
| 子概念4: 24时计时法 | "下午5时=( )时" [15:00]/[17:00]/[18:00] | 三选一快速判断 |
| 子概念1+2综合 | "一年中大月有( )个" [4]/[5]/[7]/[8] | 四选一 |

---

## 三、核心玩法循环

```
倒计时30秒开始 → 题目出现在屏幕中央
  → 看到题目和2-4个选项按钮 → 快速点击答案
  → 正确 → +10分，连击+1，连击≥5→速度加快提示
  → 错误 → -2秒惩罚，连击中断
  → 时间归零 → 结算总分，显示排名评语
```

1. **观察:** 题目闪烁出现，选项按钮排列在下方
2. **思考:** 瞬时提取知识——"7月=31天→点右边按钮"
3. **操作:** 点击或点击数字选项按钮
4. **反馈:** 正确绿色闪烁+分数飞入；错误红色抖动+扣除时间
5. **进阶:** 从二选一到三选一到四选一，题目从基础到综合

## 四、思维激发

**主要思维类型:** 快速判断、瞬时提取

**如何激发思考:**
- 速度压力迫使知识内化——必须"不假思索"反应
- 连击系统激励保持专注
- 时间惩罚让每次错误有代价，培养谨慎

## 八、游戏平衡数据

- 正确+10分，连击combo×1.2倍
- 错误-2秒时间惩罚

- 过关条件：存活到时间结束，总分≥60
