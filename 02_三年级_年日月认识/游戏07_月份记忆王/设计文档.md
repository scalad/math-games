# 游戏设计文档：月份记忆王

> **知识点:** 年、月、日的认识
> **适用年级:** 三年级下册
> **游戏机制类型:** B01 配对翻牌（卡牌族）
> **设计编号:** 2

---

## 一、核心概念

一张经典的记忆翻牌游戏！12张卡片正面朝下排列（6对配对），玩家每次翻开2张。配对规则不是找相同图片，而是找"月份 → 天数"或"月份 → 分类"的数学对应关系。例如：翻到"7月"需要配对"31天(大月)"，翻到"2月(平年)"需要配对"28天"。需要记忆+数学知识的双重挑战。

---

## 二、教学目标映射

| 教学步骤 | 游戏中的体现 | 实现方式 |
|---------|------------|---------|
| 子概念2: 大月和小月 | 配对"月份名称 ↔ 天数"或"月份 ↔ 分类标签" | 翻开两张需满足数学对应关系 |
| 子概念3: 平年和闰年 | 2月有2张不同卡片（28天/29天） | 翻到"2月(平年)"→需配对"28天"，"2月(闰年)"→"29天" |
| 子概念1: 年月日关系 | 增加"1年"↔"12个月"配对 | 建立整体关系感知 |

---

## 三、核心玩法循环

```
16张卡片面朝下排列 → 点击翻开第1张 → 看到内容(如"7月")
  → 回忆：7月是大月还是小月？有多少天？
  → 点击翻开第2张 → 如果匹配(如翻开"31天") → 配对成功！加分
  → 如果第2张不匹配(如翻开"30天") → 两张翻回，记住位置
  → 重复直到所有配对成功 → 进入下一关（更多卡片/更难配对）
```

1. **观察:** 16~20张卡片正面朝下，每张背面有"?"图案
2. **思考:** 第一次翻开看到"7月"，回忆7月属于大月=31天，再找到"31天"那张
3. **操作:** 点击卡片翻开
4. **反馈:** 配对正确卡片消失+星星特效；配对错误0.8秒后翻回
5. **进阶:** 8对(16张) → 10对(20张) → 加入闰年配对 → 加入24时配对

## 四、思维激发

**主要思维类型:** 模式识别、记忆策略

**如何激发思考:**
- 不是简单的图像记忆，必须理解"为什么这个月份对应这个天数"
- 最优策略：先翻开新卡片收集信息，还是优先配对已知卡片？
- 2月的两张卡片（28天/29天）是陷阱也是关键区分点

## 五、视觉与交互设计

**视觉风格:** 简约卡片风格，每张卡片有月份图标

**配色方案:**
- 主色：#7B68EE（中紫色）
- 辅色：#FFD700（金色）
- 强调色：#FF6B6B（珊瑚红）

**角色/元素:**
- 卡片：圆角矩形，正面有月份名称或天数数字
- 背景：柔和的星空或时间主题背景
